# TOP BPO Services - One Page Website

A professional one-page website template designed for BPO (Business Process Outsourcing) services.

## 📋 Introduction

This project is a modern, responsive one-page website designed to promote BPO services. The website showcases outsourcing solutions, customer support, call center, and data entry services.

## ✨ Key Features

- **🎨 Attractive Design**: Modern and professional UI
- **📱 Fully Responsive**: Optimized performance on all devices
- **⚡ Fast Loading**: Optimized CSS and JavaScript
- **🎭 Smooth Animations**: Revolution Slider and custom animations
- **🔧 SEO Friendly**: Meta tags and search engine optimization
- **📧 Contact Form**: Contact form with PHP backend
- **🎬 Video Support**: Video background and media support

## 🛠️ Tech Stack

### Frontend
- **HTML5**: Semantic HTML structure
- **CSS3**: Modern CSS features and animations
- **Bootstrap 4**: Responsive grid system
- **jQuery**: JavaScript library
- **Revolution Slider**: Premium slider plugin

### Backend
- **PHP**: Contact form processing

### Other Tools
- **Font Awesome**: Icon library
- **Isotope**: Filterable gallery
- **Skills Graph**: Animated skills display

## 🚀 Installation and Setup

### Requirements
- Web server (Apache, Nginx, or XAMPP)
- PHP 7.0 or higher
- Modern web browser

### Installation Steps

1. **Download and Extract**
   ```bash
   # Copy project files to your web server root
   ```

2. **Server Configuration**
   - Place files in web server root directory
   - Configure PHP (for contact form)

3. **Set Permissions**
   ```bash
   # Set execute permissions for PHP files
   chmod 755 php/contact-form.php
   ```

4. **Open in Browser**
   ```
   http://localhost/one-page-1-default/
   ```

## ⚙️ Customization

### Branding
- **Logo**: Place your logo in `images/` folder
- **Colors**: Change color variables in `css/custom.css`
- **Fonts**: Add custom fonts in `fonts/` folder

### Contact Form
- Update your email address in `php/contact-form.php`
- Customize form fields in `index.html`

### Content
- Update text content in `index.html`
- Place your images in `images/` folder

## 🔧 Troubleshooting

### Common Issues

1. **Styles Not Loading**
   - Check CSS file paths
   - Verify CSS files exist on server

2. **JavaScript Errors**
   - Check if jQuery loads first
   - Check console for errors

3. **Contact Form Not Working**
   - Check PHP server configuration
   - Verify email settings
